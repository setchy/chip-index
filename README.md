# Chip Index 🍟

> **Rate the crunch. Share the obsession.**

Chip Index is the world's first social platform built for chip lovers. Think Untappd — but for potato chips, hot chips, fries, and every starch-based snack on the planet.

## What is it?

Check in your location, log the snack you're eating, and rate it across dimensions that actually matter:

| Dimension             | What it measures                                      |
| --------------------- | ----------------------------------------------------- |
| 🌶️ **Hotness**         | Heat level — from mild to nuclear                     |
| 🫙 **Sauce**           | Dipping sauce or flavour sauce quality                |
| 🧂 **Seasoning**       | Flavour depth and accuracy                            |
| ✨ **Crunch Factor**   | The all-important texture                             |
| 💰 **Value for Money** | Chip-to-price ratio — hidden gems vs. highway robbery |
| 🔁 **Repeatability**   | Would you buy this again? The true overall signal     |
| 🍺 **Drinkability**    | How well it pairs with your drink of choice           |

Share reviews with friends, discover hidden gems around the world, and build your snack adventure history.
